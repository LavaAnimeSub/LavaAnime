# 关于熔岩动画

**标有 熔岩动画搬运（LavaAnime）** 意为此字幕的翻译通常来自官方，仅增加了一些样式和微调，请自行判断能否接受。

**标有 熔岩动画Sub（LavaAnimeSub）** 意为此字幕作品的翻译来源是本组或是已授权本组使用的译源。是偶尔可能出现的别名，区别于上述的搬运组。
